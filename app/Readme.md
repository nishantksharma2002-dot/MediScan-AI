# MediScan-AI

## 📌 Description
MediScan-AI is a backend system for analyzing medical data using AI.

---

## ⚙️ Tech Stack
- Python
- FastAPI (or Flask)
- SQL Database

---

## 📂 Project Structure
```
app/
  models/
  routes/
  schemas/
  services/
  utils/
  main.py
```

---

## 🚀 How to Run

### 1. Clone repo
```
git clone <your-repo-url>
cd MediScan-AI
```

### 2. Install dependencies
```
pip install -r requirements.txt
```

### 3. Run server
```
uvicorn app.main:app --reload
```
